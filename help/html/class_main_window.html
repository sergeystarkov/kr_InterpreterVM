MainWindow::on_startInterpereter_clicked - Кнопка запуска инретпретатора

Сохраняет текст программы из редактора во временный файл. Инициализирует поток для интерпретатора. Запскает интерпретатор в отдельном от GUI потоке (в качестве файла программы выступает временный *.tmp файл). По завершении работы интерпретатора удаляет временый файл.
